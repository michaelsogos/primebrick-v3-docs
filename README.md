# Primebrick Docs

Documentation site for [Primebrick](https://primebrick.dev) — built with [Zudoku](https://zudoku.dev) and deployed to Cloudflare Workers as a static site.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:8787](http://localhost:8787) with your browser to see the result.

Edit pages in the `pages/` directory — content updates automatically via HMR.

## Build

```bash
pnpm build
```

Output is written to `dist/`. The build runs sync scripts to fetch OpenAPI specs and DeepWiki content from other Primebrick repos.

## Deploy

```bash
pnpm deploy
```

Deploys to Cloudflare Workers using `wrangler`.

## Structure

```
pages/              Documentation pages (MDX/MD)
apis/               OpenAPI specs (generated by scripts/fetch-openapi.mjs)
scripts/            Build-time sync and generation scripts
src/generated-*.ts  Auto-generated nav + API config (do not edit)
zudoku.config.tsx   Zudoku configuration
wrangler.jsonc      Cloudflare Workers config
```
